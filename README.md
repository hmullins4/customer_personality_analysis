# customer_personality_analysis

This project analyzed a customer marketing dataset to understand which customers were most likely to respond to a campaign and to explore whether meaningful customer segments could be identified through clustering. Using Python and scikit-learn, I cleaned the data, handled missing values, encoded categorical features, and engineered variables related to customer demographics and purchasing behavior. I then applied unsupervised learning methods such as K-means, PCA-based clustering, Gaussian Mixture Models, and DBSCAN to evaluate segmentation patterns, and built several supervised classification models, including logistic regression, decision tree, random forest, and SVM, to predict campaign response. To address class imbalance, I used techniques such as SMOTE and class weighting, and the best-performing RBF-kernel SVM model achieved very high test accuracy in predicting whether a customer would accept the campaign.
